Configure you browser/OS proxy settings for proper Internet access

Internet Exporer and Chrome use OS settings that can be changed in Control Panel,
Firefox has own proxy settings in 'Preferences/Advanced/Network/Settings'

Enter these parametars for manual proxy configuration:
HTTP Proxy: {{ PROXY_HOSTNAME }}.{{ PROXY_DOMAIN }}
Port: 8080
Select 'Use for all protocols'

or

for Automatic proxy configuration:
URL: http://{{ PROXY_HOSTNAME }}.{{ PROXY_DOMAIN }}/proxy.pac
